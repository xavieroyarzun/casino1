Aplicación

Esta página es un proyecto educativo diseñado para aprender y poner en práctica el uso de HTML, CSS y JavaScript. La aplicación desarrollada es una ruleta digital, en la cual se utilizó inteligencia artificial como principal herramienta tecnológica, tanto para la generación de imágenes como para brindar asistencia durante el proceso de creación. El resultado es una aplicación funcional y versátil, que aún puede mejorarse en futuras versiones.
